Installing the Mirabilis Skin

Current The Rascal REPO Version: 3.1.0

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://therascal.github.io/repo
  • Name it ".therascal" without the quotes
  • Head to "My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select ".therascal", and install repository.therascal-3.1.0.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the ".therascal" menu item
  • Open "Look-and-Feel" - "Skin" and install Mirabilis

Enjoy!
